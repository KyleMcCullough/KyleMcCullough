Instructions on how to run the application.

1. Open the server folder and run ChatConsole.exe in a console with the argument -server at the end.
2. Open Visual Studio project file to run application OR go to Assignment2\bin\Debug and open Assignment2.exe

Basic Controls

GUI

Network->Connect : Connects to the server
Network->Disconnect : Disconnects from the server
Game->Quit: Disconnects from the server if still connected and exits the application.
Send : Sends the message in the box on the left to the server.


CLI

I : Enters insert mode.
"quit" : Typing quit closes the server.
